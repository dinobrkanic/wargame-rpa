class wargame.hut.Hut(number, occupant)¶

Osnovice: object

Klasa koja prezentira kucicu.

Parametri

  • number (int) – broj kucice

  • occupant (AbstractGameUnit) – Tip okupanta kucice

Varijable

  • number (int) – broj pridruzen kucici

  • is_acquired (boolean) – je li kucica osvojena od igraca Taliona

  • occupant (AbstractGameUnit) –

    Okupant kucice.

    Mora biti instanca klase

    AbstractGameUnit.

Više informacija

gdje se još koristi – attackoftheorcs.AttackOfTheOrcs.setup_game_scenario()

get_occupant_type()¶

Vraca tekst s opisom o vrsti zauzeća kućice.

Koristi se za ispis informacije tko je prisutan u kućici. Informaciju koju vrati ovisi o okupantu i može biti: ‘neprijatelj’, ‘prijatelj’, ‘ZAUZETA’,’slobodna’.

Vraća

string koji predstavlja vrstu okupanta
